# 《三命通会》智能命理分析系统

## 1. 功能概述

本 Skill 基于明代万民英所著的命理经典《三命通会》，提供完整的八字命理分析能力。

### 1.1 核心功能

| 功能 | 触发关键词 | 描述 |
|------|----------|------|
| **八字排盘** | 算命、排盘、八字 | 根据出生年月日时计算完整八字 |
| **命理分析** | 分析命运、人生、性格 | 基于《三命通会》进行深度命理分析，白话解释 |
| **事件预测** | 好不好、能否、应不应该 | 分析特定事件的吉凶 |
| **知识问答** | 什么是、解释、原理 | 解答命理知识问题 |

### 1.2 触发条件

当用户使用以下表达时，应激活本 Skill：

- "我想算命" / "帮我看看命"
- "分析一下八字"
- "明天签合同好不好"
- "这个月运势如何"
- "什么是七杀"
- "甲木日主命运如何"
- 任何关于八字、命理、风水、择日的询问

## 2. 工具说明

### 2.1 八字排盘工具 (bazi_paipan.py)

**文件位置**: `sanmingtonghui-skill/scripts/bazi_paipan.py`

**用途**: 将出生时间转换为八字（年柱、月柱、日柱、时柱），并计算五行、十神等信息。

#### 调用方式

**命令行调用**:
```bash
cd sanmingtonghui-skill/scripts
python bazi_paipan.py --year 1990 --month 1 --day 1 --hour 12 --gender 男
```

**参数说明**:

| 参数 | 类型 | 必须 | 说明 | 示例 |
|------|------|------|------|------|
| `--year` | int | 是 | 公历年份 | 1990 |
| `--month` | int | 是 | 月份 (1-12) | 1 |
| `--day` | int | 是 | 日期 (1-31) | 1 |
| `--hour` | int | 是 | 小时 (0-23) | 12 |
| `--gender` | str | 是 | 性别 | 男/女 |
| `--lunar` | flag | 否 | 表示输入为农历时间 | --lunar |

**输出格式**:
```json
{
  "bazi": {
    "year": {"gan": "庚", "zhi": "午", "gan_wuxing": "金", "zhi_wuxing": "火"},
    "month": {"gan": "辛", "zhi": "丑", "gan_wuxing": "金", "zhi_wuxing": "土"},
    "day": {"gan": "丙", "zhi": "申", "gan_wuxing": "火", "zhi_wuxing": "金"},
    "hour": {"gan": "甲", "zhi": "午", "gan_wuxing": "木", "zhi_wuxing": "火"}
  },
  "bazi_string": "庚午 辛丑 丙申 甲午",
  "wuxing": {"木": 1, "火": 3, "土": 2, "金": 2, "水": 0},
  "shishen_detail": {
    "year_shishen": {"gan": "庚", "zhi": "午", "shishen": "官", "gan_wuxing": "金", "zhi_wuxing": "火"},
    "month_shishen": {"gan": "辛", "zhi": "丑", "shishen": "才", "gan_wuxing": "金", "zhi_wuxing": "土"},
    "day_shishen": {"gan": "丙", "zhi": "申", "shishen": "比", "gan_wuxing": "火", "zhi_wuxing": "金"},
    "hour_shishen": {"gan": "甲", "zhi": "午", "shishen": "印", "gan_wuxing": "木", "zhi_wuxing": "火"}

    "solar_date": "1990-1-1",
    "lunar_date": "1989-11-15",
```

**Python 模块调用**:
```python
from bazi_paipan import BaziPaipan

paipan = BaziPaipan()
result = paipan.paipan(1990, 1, 1, 12, is_lunar=False, gender="男")
print(result["bazi_string"])  # 庚午 辛丑 丙申 甲午
```

**直接输入八字字符串**:
```python
result = paipan.paipan_from_string("庚午 辛丑 丙申 甲午", gender="男")
```

#### 使用时机

- 用户提供出生年月日时后
- 用户询问特定时间的八字时
- 需要进行命理分析前的数据准备

### 2.2 RAG 检索工具 (rag_retriever.py)

**文件位置**: `sanmingtonghui-skill/scripts/rag_retriever.py`

**用途**: 从《三命通会》知识库中检索与当前八字或问题相关的原文片段。

**特点**: 零外部依赖，基于纯关键词匹配检索，无需 Embedding 模型。

#### 调用方式

**命令行调用**:
```bash
cd sanmingtonghui-skill/scripts
python rag_retriever.py --query "甲木日主生在寅月" --top_k 5
```

**参数说明**:

| 参数 | 类型 | 必须 | 说明 | 示例 |
|------|------|------|------|------|
| `--query` | str | 是 | 查询文本 | 甲木日主 |
| `--top_k` | int | 否 | 返回结果数量，默认5 | 5 |
| `--kb_path` | str | 否 | 知识库路径 | ../data/sanmingtonghui_chunks.json |

**输出格式**:
```json
[
  {
    "text": "甲木喜用丁火，丙火次之。...甲木参天，脱胎要火...",
    "source": "卷四·论甲木",
    "relevance": 0.892,
    "chunk_id": 156
  },
  {
    "text": "寅月甲木，上半月专用火...下半月虽用火，亦宜兼用土...",
    "source": "卷四·甲木生于寅月",
    "relevance": 0.856,
    "chunk_id": 157
  }
]
```

**Python 模块调用**:
```python
from rag_retriever import RAGRetriever

retriever = RAGRetriever("sanmingtonghui-skill/data/sanmingtonghui_chunks.json")
results = retriever.retrieve("甲木日主", top_k=5)

for r in results:
    print(f"来源: {r['source']}")
    print(f"相关性: {r['relevance']:.3f}")
    print(f"内容: {r['text'][:100]}...")
    print()
```

**按八字自动检索**:
```python
results = retriever.retrieve_by_bazi(bazi_info, query="事业财运", top_k=5)
```

#### 使用时机

- 完成八字排盘后，需要检索《三命通会》原文进行分析时
- 用户询问具体命理问题时
- 进行事件预测时检索相关理论依据时

```
    ├── 白话解释命运特点
```

## 5. 关键概念

### 5.1 五行 (Wuxing)

五行是命理的根基，包括木、火、土、金、水五种基本元素。

| 五行 | 代表事物 | 特性 | 方位 | 季节 |
|------|---------|------|------|------|
| 木 | 东方、青色、生 | 生发、条达 | 东 | 春 |
| 火 | 南方、赤色、长 | 炎热、向上 | 南 | 夏 |
| 土 | 中央、黄色、化 | 长养、化育 | 中 | 四季末 |
| 金 | 西方、白色、收 | 收敛、肃杀 | 西 | 秋 |
| 水 | 北方、黑色、藏 | 寒冷、流动 | 北 | 冬 |

**相生**: 木→火→土→金→水→木
**相克**: 木克土→土克水→水克火→火克金→金克木

### 5.2 天干 (Tianggan)

十天干代表天象运行的十种能量：

| 天干 | 五行 | 阴阳 | 方位 |
|------|------|------|------|
| 甲 | 木 | 阳 | 东北 |
| 乙 | 木 | 阴 | 东北 |
| 丙 | 火 | 阳 | 东南 |
| 丁 | 火 | 阴 | 东南 |
| 戊 | 土 | 阳 | 南 |
| 己 | 土 | 阴 | 南 |
| 庚 | 金 | 阳 | 西南 |
| 辛 | 金 | 阴 | 西南 |
| 壬 | 水 | 阳 | 西北 |
| 癸 | 水 | 阴 | 西北 |

**天干相生**: 甲乙木生丙丁火 → 丙丁火生戊己土 → 戊己土生庚辛金 → 庚辛金生壬癸水 → 壬癸水生甲乙木
**天干相克**: 甲乙木克戊己土 → 丙丁火克庚辛金 → 戊己土克壬癸水 → 庚辛金克甲乙木 → 壬癸水克丙丁火

### 5.3 地支 (Dizhi)

| 午 | 火 | 阳 | 马 | 五月 |
| 未 | 土 | 阴 | 羊 | 六月 |
| 申 | 金 | 阳 | 猴 | 七月 |
| 酉 | 金 | 阴 | 鸡 | 八月 |
| 戌 | 土 | 阳 | 狗 | 九月 |
| 亥 | 水 | 阴 | 猪 | 十月 |

### 5.4 十神 (Shishen)

十神是根据日干与其他天干的关系推算出来的符号：

| 十神 | 计算方式 | 阴阳 | 特性 |
|------|---------|------|------|
| 比肩 | 同我者同阴阳 | 阳见阳/阴见阴 | 刚健、自立、竞争 |
| 劫财 | 同我者异阴阳 | 阳见阴/阴见阳 | 刚强、争斗、破耗 |
| 食神 | 我生者同阴阳 | 阳见阴/阴见阳 | 食禄、才华、享乐 |
| 伤官 | 我生者异阴阳 | 阳见阳/阴见阴 | 才华、叛逆、创新 |
| 偏财 | 我克者同阴阳 | 阳见阳/阴见阴 | 财运、慷慨、流动性 |
| 正财 | 我克者异阴阳 | 阳见阴/阴见阳 | 财运、勤俭、稳定性 |
| 七杀 | 克我者同阴阳 | 阳见阳/阴见阴 | 权威、压力、变革 |
| 正官 | 克我者异阴阳 | 阳见阴/阴见阳 | 地位、规矩、责任 |
| 偏印 | 生我者同阴阳 | 阳见阳/阴见阴 | 学术、偏门、孤僻 |
| 正印 | 生我者异阴阳 | 阳见阴/阴见阳 | 学识、庇护、仁慈 |

**十神与六亲**:

| 十神 | 代表六亲 | 性别(以日干论) |
|------|---------|--------------|
| 比肩、劫财 | 兄弟姐妹 | 同性为比肩，异性为劫财 |
| 食神、伤官 | 子女 | 同性为食神，异性为伤官 |
| 正财、偏财 | 妻财/父亲 | 男命为妻，女命为父 |
| 正官、七杀 | 官禄/丈夫/子女 | 男命为官/子女，女命为夫 |
| 正印、偏印 | 母亲/祖父 | 同性为偏印，异性为正印 |

### 5.5 神煞 (Shensha)

神煞是命理中重要的辅助判断符号：

#### 吉神

| 神煞 | 含义 | 应用 |
|------|------|------|
| 天乙贵人 | 最吉之神 | 有助无难，遇事有人帮 |
| 文昌 | 文采之星 | 利学业、考试 |
| 太极贵人 | 聪明有德 | 有哲思，精神追求 |
| 三奇贵人 | 特殊吉祥 | 特殊命格 |
| 国印贵人 | 掌印之星 | 利仕途、管理 |

#### 凶煞

| 神煞 | 含义 | 应用 |
|------|------|------|
| 羊刃 | 刚暴之星 | 刚强过甚，易有灾祸 |
| 驿马 | 走动之星 | 奔波、变动、外出 |
| 桃花 | 感情之星 | 感情、桃花运 |
| 空亡 | 虚无之星 | 做事难成，白费力气 |
| 灾煞 | 意外灾祸 | 意外、血光 |
| 勾绞 | 纠纷之星 | 口舌、纠纷 |

### 5.6 大运与流年

**大运 (Dayun)**: 从命主出生起，每十年一个运程，影响人生不同阶段的运势。

**流年 (Liunian)**: 每年的年柱干支，反映该年的运势变化。

**起运规则**: 
- 男命阳年出生顺行，阴年出生逆行
- 女命阴年出生顺行，阳年出生逆行
- 通常三岁或八岁起运

## 7. 注意事项

### 7.1 信息收集注意事项

1. **时间准确性**: 八字排盘依赖准确的出生时间，误差可能导致八字完全错误
2. **公历/农历区分**: 必须确认用户提供的是公历还是农历时间
3. **真太阳时**: 中国幅员辽阔，东部和西部存在时差，高质量分析需考虑真太阳时
4. **性别区分**: 大运起运规则男女不同，必须确认性别

### 7.2 分析判断注意事项

1. **综合判断**: 命理分析需综合考量多个因素，不可单凭一点下结论
2. **原文为据**: 所有分析结论应参考《三命通会》原文作为依据
3. **用神取法**: 用神的取法有多种，应根据具体命局灵活判断
4. **大运流年**: 应结合大运和流年进行动态分析

### 7.3 白话解释原则

1. **避免术语堆砌**: 尽量少用"身旺""身弱""用神""忌神"等专业术语，必须使用时立即解释
2. **生活化比喻**: 用日常事物比喻命理概念（如"火多就像脾气急""水少就像容易口干"）
3. **现代场景结合**: 结合职场、婚恋、健康等现代生活场景解释
4. **积极建设性**: 即使命局有不足，也要给出改善建议，避免危言耸听
5. **具体可操作**: 建议要具体，如"适合往北方发展"而非"宜水"

### 7.4 结果表达注意事项

1. **语言得体**: 分析语言应专业但温和，不可危言耸听
2. **积极引导**: 即使命局有不足，也应给出改善建议
3. **明确说明**: 分析仅供参考，不可作为重大决策的唯一依据
4. **保护隐私**: 不泄露用户提供的任何个人信息

### 7.5 伦理规范

1. **适度分析**: 命理分析应适度，不可过度依赖
2. **正向引导**: 引导用户积极面对命运，而非消极等待
3. **专业边界**: 命理分析仅供参考，不可替代专业咨询（如医疗、法律等）
4. **文化传承**: 尊重命理文化传统，传承中华智慧
