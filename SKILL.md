# 《三命通会》智能命理分析系统

## 1. 功能概述

本 Skill 基于明代万民英所著命理经典《三命通会》，提供完整的八字命理分析能力。

### 1.1 核心功能

| 功能 | 触发关键词 | 描述 |
|------|----------|------|
| **八字排盘** | 算命、排盘、八字、四柱 | 根据出生年月日时计算完整八字 |
| **大运流年** | 大运、流年、运势、运程 | 十年大运排列和流年分析 |
| **神煞判断** | 神煞、贵人、桃花、文昌 | 吉神凶煞自动判断 |
| **命宫胎元** | 命宫、胎元、身宫 | 辅助命宫盘计算 |
| **风水参考** | 风水、方位、颜色、行业 | 喜用神与风水建议 |
| **择日分析** | 择日、吉日、良辰、好不好 | 特定日期事件吉凶分析 |
| **合盘分析** | 合盘、合婚、配对、八字合不合 | 两人八字对比分析 |
| **原文检索** | 三命通会、原文、典籍 | 从知识库检索《三命通会》原文 |

### 1.2 触发条件

当用户使用以下表达时，应激活本 Skill：

- "我想算命" / "帮我看看八字"
- "分析一下我的命运"
- "明天结婚好不好" / "这个日子吉利吗"
- "我和TA八字合不合"
- "我的大运怎么样"
- "什么是七杀" / "解释一下十神"
- "我适合什么行业" / "我的喜用神是什么"
- 任何关于八字、命理、风水、择日、合盘的询问

### 2.2 八字排盘库 (bazi_paipan.py)

**文件位置**: `scripts/bazi_paipan.py`

**用途**: 核心排盘库，提供所有命理计算方法。

#### Python 调用

```python
from bazi_paipan import BaziPaipan
from lunar_python import Solar

paipan = BaziPaipan()

# 基础排盘
result = paipan.paipan(1990, 5, 15, 14, gender="男")

# 从八字字符串排盘
result = paipan.paipan_from_string("庚午 辛巳 庚辰 癸未", gender="男")

# 大运
solar = Solar.fromYmdHms(1990, 5, 15, 14, 0, 0)
lunar = solar.getLunar()
qiyun = paipan.compute_qiyun(solar, lunar, "男")
dayun = paipan.compute_dayun(result["bazi"], "男")

# 流年
liunian = paipan.compute_liunian(2026)

# 神煞
shensha = paipan.compute_shensha(result["bazi"])

# 命宫胎元身宫
minggong = paipan.compute_minggong(result["bazi"])
taiyuan = paipan.compute_taiyuan(result["bazi"])
shengong = paipan.compute_shengong(result["bazi"])

# 风水
fengshui = paipan.fengshui_reference(result["bazi"])

# 择日
zeri = paipan.zeri_analysis(2025, 6, 1, event_type="结婚")

# 合盘
bazi2 = paipan.paipan(1992, 3, 8, 10, gender="女")
hepan = paipan.hepan_analysis(result["bazi"], bazi2["bazi"])
```

## 5. 关键概念速查

### 5.1 五行

| 五行 | 特性 | 方位 | 季节 |
|------|------|------|------|
| 木 | 生发、条达 | 东 | 春 |
| 火 | 炎热、向上 | 南 | 夏 |
| 土 | 长养、化育 | 中 | 四季末 |
| 金 | 收敛、肃杀 | 西 | 秋 |
| 水 | 寒冷、流动 | 北 | 冬 |

**相生**: 木→火→土→金→水→木
**相克**: 木克土→土克水→水克火→火克金→金克木

### 5.3 神煞

## 6. 注意事项

### 6.1 信息准确性

1. **时间准确性**: 八字排盘依赖准确的出生时间
2. **公历/农历区分**: 必须确认用户提供的是公历还是农历
3. **真太阳时**: 中国东西部存在时差，高质量分析需考虑
4. **性别区分**: 大运起运规则男女不同

### 6.2 分析原则

1. **综合判断**: 不可单凭一点下结论
2. **原文为据**: 参考《三命通会》原文作为依据

4. **积极建设**: 即使命局有不足，也要给出改善建议
